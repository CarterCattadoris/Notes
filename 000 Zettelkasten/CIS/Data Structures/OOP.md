**Object-Oriented Programming (OOP)** is the practice of organizing code into objects and classes.

## Core Concepts

- [[Classes]] - Templates for objects
- [[Object]] - Instances of classes
- [[Inheritance]] - Subclass relationships
- [[Polymorphism]] - Multiple behaviors for same interface
- [[Abstract Classes]] - Base classes that cannot be instantiated
