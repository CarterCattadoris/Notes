Let $P = (A, \preceq)$ be a poset.  
- An element w ∈ A is maximal if there is no $b ∈ A$ such that $w ≺ b$.  
- Note: $w ≺ b$ iff ($w \preceq b$ and $b \neq w$ ).  
An element $w ∈ A$ is minimal if there is no $a ∈ A$ such that $a ≺ w$
