## Set Operations: 
S is a set, X changes
- Membership: $X \in S$ 
- [[Cardinality]]: |S|
- [[Subsets]]: $X \subseteq S$ 
	- [[Powersets]]: $X \in 2^S$ 
- [[Intersection]]: $X \cap S$
- [[Unions]]: $X \cup S$ 
- [[Set Equality]]: X = S
- [[Set Difference]]: S - X
- [[Symmetric Difference]]: $X \triangle S$
- [[Cartesian Product]]: $X \times S$
